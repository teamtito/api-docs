1st March 2021

Added

Updated

  • Release endpoints have deprecated payment_type in favour of the new pricing_type attribute which is not read-only. To change the pricing type set it to either free, paid, donation or degressive.

Added

  • Add Webhook Endpoint API docs
  • Add Headers section to specify which headers are required to be sent with requests.

19th August 2019

Added

  • Added Webhook JSON payloads for checkin.* webhook types.

5th August 2019

Added

  • Added Webhook JSON payloads for ticket.* and registration.* webhook types.

31st July 2019

Added

  • Added docs for receiving webhooks from Tito to your local development environment.

6th March 2019

Modified

When we return the metadata for an event, release, registration or ticket we now return an object rather than a JSON string. So instead of seeing "metadata": "{\"foo\":\"bar\"}" you will see "metadata":{"foo":"bar"}".

Modified
